# Fitbit Activity, Heart Rate, and Sleep Analysis

This project explores relationships between daily activity, heart rate, and sleep using Fitbit data.

## 🔍 Key Questions
- Do higher activity levels lead to higher heart rate?
- Does step count correlate with heart rate?
- Is sleep duration related to heart rate or activity?

## 🧰 Tools Used
- Python
- pandas
- matplotlib
- seaborn

## 📊 Key Findings
- Step count alone is not a strong predictor of heart rate
- Categorizing activity levels revealed clearer patterns
- Heart rate generally increases with higher activity levels
- Sleep duration does not show a clear relationship with heart rate
- No meaningful relationship between steps and sleep duration
